# 🤖 Agent IA Multitool - LangChain

Un assistant IA personnel capable de :
- Lire des documents (PDF) et répondre à des questions (RAG)
- Faire des recherches web
- Gérer une TODO list
- Répondre à des questions générales et calculs
- Garder un fil conversationnel cohérent
